# vedha-web

Website code for Vedha - a tech and marketing agency.

## Project info

This is the website codebase for Vedha, built with modern web technologies.

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Framer Motion

## Getting Started

### Prerequisites

- Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```sh
# Clone the repository
git clone https://github.com/yashbanerjee/vedha-web.git

# Navigate to the project directory
cd vedha-web

# Install dependencies
npm install

# Start the development server
npm run dev
```

## Features

- Dark/Light theme toggle
- Responsive design
- Smooth animations and transitions
- Interactive scrolling problem text with hover effects
- Modern UI with custom dot grid background
